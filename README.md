# 愚公新传：移山志
> 新兴神话故事，由你来见证

一款基于 Unity 开发的国风放置点击 + 模拟经营休闲游戏，改编自传统神话《愚公移山》，讲述愚公带领村民开山辟路、子子孙孙无穷匮也的故事。

---

## 🎮 游戏介绍
《愚公新传：移山志》是一款 12+ 休闲放置类游戏，主打轻松养成、资源管理与国风剧情体验。
- **游戏类型**：放置点击 / 模拟经营 / 国风叙事
- **开发引擎**：Unity
- **单局时长**：10–15 分钟
- **核心主题**：改编自传统神话，以现代视角诠释“愚公移山”的坚持与传承精神
- **目标玩家**：喜欢轻松养成、策略规划、国风题材的休闲玩家

---

## ✨ 游戏特色
- **轻松放置，自动成长**：放置也能自动积累资源，零压力养成
- **策略经营，多线发展**：生产力、工具、人口三大资源协调分配，规划最优移山路线
- **关卡递进，挑战升级**：五座大山逐级挑战，山体血量随进度递增，成长反馈清晰
- **剧情沉浸，国风叙事**：完整主线剧情，随移山进度逐步解锁，还原神话内核
- **大数值成长系统**：自定义 `StringBigInt` 超大数值类，满足长期养成的成就感
- **无强制流程，高自由度**：佛系也能玩，没有硬性任务压力
- **数据持久化**：跨场景存档，进度随时保存

---

## 🎯 核心玩法
1.  **点击采集 + 放置收益**
    点击山体采集资源，放置状态下也能自动积累，轻松推进进度。
2.  **资源管理与升级**
    消耗资源生产工具、招募村民、提升生产力，获得更高的移山效率。
3.  **移山挑战与关卡推进**
    每座山都有独立的血量与挑战目标，凿开大山即可解锁新场景与剧情。
4.  **剧情分支与叙事体验**
    随着移山进度推进，逐步解锁愚公与村民的故事，感受传统神话的现代演绎。

## 🛠️ 技术亮点
- **自定义大数值系统**：实现 `StringBigInt` 超大数值类，支持无上限数值增长与显示
- **模块化系统设计**：山体血量控制、对话系统、事件驱动架构，降低模块耦合度
- **性能优化**：对象池技术减少GC压力，多场景数据持久化保证存档安全
- **用户体验优化**：自适应UI布局、亮度调节设置、离线收益机制


## 📥 下载与游玩（Windows）
游戏可执行包已发布在本仓库 **Releases**，下载即可直接游玩：

1. 前往右侧 [Releases](https://github.com/nikon1koni/The-New-Legend-of-the-Foolish-Old-Man/releases) 下载最新版压缩包
2. 解压到**纯英文路径**（不要有中文、空格、特殊符号）
3. 打开解压文件夹，双击 `愚公新传：移山志.exe` 运行游戏

### ⚠️ 重要注意
- 仅支持 **Windows 64 位系统**
- 必须保持 `exe`、`xxx_Data` 文件夹、`UnityPlayer.dll` 在同一目录，不可拆分移动
- 游戏进度自动存档，重启可继续游玩
